**현재 흐름 (구버전, 이미지 기준):**
```
$SITE_STATUS:1@         →
                        ← $SITE_STATUS:1,READY@
$LIGHT:1,1,1@           →
                        ← $LIGHT:1,1@         ← 삭제 대상
$TEST:1,TOP,1,null,0@   →                     ← z_index 제거 대상
                        ← $RESULT:1;TOP;B;0;@
$TEST:1,TOP,1,null,1@   →
                        ← $RESULT:1;TOP;B;3;A1=OK,A2=OK,A3=OK@
...
$LIGHT:1,1,0@           →
                        ← $LIGHT:1,0@         ← 삭제 대상
```

**수정 후 흐름 (v3.0 신규):**
```
$SITE_STATUS:1@         →
                        ← $SITE_STATUS:1,READY@
$LIGHT:1,1,1@           →   (응답 없음)
$PREP:1,0@              →                     ← z_index 선행 (신규)
                        ← $PREP_ACK:1,0,OK@   ← (신규)
$TEST:1,TOP,1@          →                     ← z_index 제거됨
                        ← $RESULT:1;TOP;B;0;@
$PREP:1,1@              →
                        ← $PREP_ACK:1,1,OK@
$TEST:1,TOP,1@          →
                        ← $RESULT:1;TOP;B;3;A1=OK,A2=OK,A3=OK@
... (Index마다 $PREP→$PREP_ACK→$TEST 반복)
$LIGHT:1,1,0@           →   (응답 없음)
```

**핵심 변경 3가지:**
1. `$LIGHT` 응답 라인(`← $LIGHT:1,1@`, `← $LIGHT:1,0@`) **삭제**
2. 각 Index마다 `$TEST` 앞에 `$PREP:site,z_index@` → `$PREP_ACK:site,z_index,OK@` **추가**
3. `$TEST` 포맷에서 z_index 필드 **제거** (`$TEST:1,TOP,1,null,0@` → `$TEST:1,TOP,1@`)
