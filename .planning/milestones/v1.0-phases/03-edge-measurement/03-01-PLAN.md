**2. FAIEdgeMeasurementService.cs 생성** (`WPF_Example/Halcon/Algorithms/`)

namespace `ReringProject.Halcon.Algorithms`. Allman brace style.

핵심 메서드: `public bool TryMeasure(HImage image, FAIConfig fai, out FAIEdgeMeasurementResult result)`

구현 로직:
1. null 체크 + ROI 유효성 검사 (ROI_Length1 > 0 && ROI_Length2 > 0).
2. `image.GetImageSize(out imageWidth, out imageHeight)`.
3. FAIConfig의 ROI 파라미터로 직접 `GenMeasureRectangle2` 호출:
   - Row = `fai.ROI_Row`, Col = `fai.ROI_Col`, Phi = `fai.ROI_Phi`
   - HalfLength1 = `fai.ROI_Length1`, HalfLength2 = `fai.ROI_Length2`
   - ToRoiDefinition() 우회 — center+phi 파라미터 직접 사용으로 정밀도 유지.
4. `MeasurePos` 호출: Sigma = `Math.Max(0.4, fai.Sigma)`, Threshold = `Math.Max(1, fai.Threshold)`, Transition = `"all"` (양방향), Select = `"all"` (모든 에지 추출).
5. `CloseMeasure(handle)` 반드시 호출 (try/finally).
6. 에지 개수 검사: `rows.TupleLength() < 2` 이면 false 반환 (에지 페어 필요).
7. MeasureType에 따라 에지 인덱스 선택:
   - `FirstToFirst`: idx1=0, idx2=0 (단일 에지 — 거리=0이 되므로 실용적으로 idx1=0, idx2=1)
   - `FirstToLast`: idx1=0, idx2=rows.TupleLength()-1
   - `LastToFirst`: idx1=rows.TupleLength()-1, idx2=0
   - `LastToLast`: idx1=rows.TupleLength()-1, idx2=rows.TupleLength()-1 (idx1=len-2, idx2=len-1)
   NOTE: FirstToFirst와 LastToLast는 논리적으로 동일 에지를 가리킬 수 있으므로, 실질적 구현은:
   - `FirstToFirst` / `FirstToLast`: idx1=0 고정, idx2 = MeasureType==FirstToFirst ? Math.Min(1, len-1) : len-1
   - `LastToFirst` / `LastToLast`: idx2=len-1 고정, idx1 = MeasureType==LastToFirst ? 0 : Math.Max(len-2, 0)
8. 에지 좌표 추출: `edge1Row = rows[idx1].D`, `edge1Col = cols[idx1].D`, `edge2Row = rows[idx2].D`, `edge2Col = cols[idx2].D`.
9. 픽셀 거리 계산: `double pixelDist = Math.Sqrt(Math.Pow(edge2Row - edge1Row, 2) + Math.Pow(edge2Col - edge1Col, 2))`.
10. mm 변환: ROI_Phi가 0(수평 스캔)이면 `mmDist = Math.Abs(edge2Col - edge1Col) * fai.PixelResolutionX`, ROI_Phi가 PI/2(수직 스캔)이면 `mmDist = Math.Abs(edge2Row - edge1Row) * fai.PixelResolutionY`. 일반적: `mmDist = pixelDist * fai.PixelResolutionX` (등방성 해상도 가정, 대부분의 산업용 렌즈).
11. 오버레이 생성 — 3개의 EdgeInspectionOverlay:
    - 에지 1 마커: RoiId = `"FAI-Edge1"`, Line = ROI 전체 높이에 걸친 수직/수평선 at edge1 position.
    - 에지 2 마커: RoiId = `"FAI-Edge2"`, Line = 동일 패턴 at edge2 position.
    - 연결선: RoiId = `"FAI-DistLine"`, Line = (edge1Row, edge1Col) → (edge2Row, edge2Col).
12. result 객체 조립 후 true 반환.
