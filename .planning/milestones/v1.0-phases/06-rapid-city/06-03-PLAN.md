   case EStep.Measure:
     var parentSeq2 = Param.Parent as InspectionSequence;
     bool allPass = true;
     if (ShotParam != null) {
         using (var image = ShotParam.GetImage()) {
             if (image != null) {
                 foreach (var fai in ShotParam.FAIList) {
                     foreach (var meas in fai.Measurements) {
                         HTuple transform;
                         if (parentSeq2 != null) {
                             parentSeq2.TryGetDatumTransform(meas.DatumRef, out transform);

                             HOperatorSet.HomMat2dIdentity(out transform);

                         double resultMm;

                             meas.LastJudgement = false;
                         }
                         if (!meas.LastJudgement) allPass = false;
                     }
                 }
             }
         }
     }
     pMyContext.AllPass = allPass;
     Step = (int)EStep.End;
     break;

InspectionRecipeManager의 Save/Load를 Phase 6 INI 포맷으로 전면 재작성한다 (per D-17, D-22, RC-05).

1. **포맷 버전 감지 로직 추가:**
   - `private enum ERecipeFormatVersion { Unknown = 0, Phase5 = 5, Phase6 = 6 }`
   - `private ERecipeFormatVersion DetectFormatVersion(IniFile iniFile)`:
     - `iniFile`에 "FORMAT" 섹션이 있고 "Version" 키가 6 이상이면 → Phase6
     - "SHOTS" 섹션이 있지만 "FORMAT" 섹션이 없으면 → Phase5
     - 둘 다 없으면 → Unknown

2. **Load 메서드 수정:**
   - 진입점에서 `DetectFormatVersion(loadFile)` 호출
   - Phase5 또는 Unknown → `CustomMessageBox.Show("안내", "이 레시피는 이전 포맷(Phase 1~5)입니다.\n새 레시피로 작성하세요.", MessageBoxImage.Information)` 표시 후 `return false` (D-22)
   - Phase6 → `LoadPhase6Format(loadFile)` 호출

3. **LoadPhase6Format(IniFile loadFile) 구현:**
   INI 구조 (RESEARCH.md 참고):
   ```
   [FORMAT] Version=6
   [FIXTURE] DisplayName=Top 면, DatumCount=2
   [FIXTURE_DATUM_0] DatumName=B, ImageSourceMode=Dedicated, ... (DatumConfig 필드)
   [FIXTURE_DATUM_1] ...
   [SHOTS] Count=2
   [SHOT_0] ShotName=SHOT_0, ZPosition=0, RingLight_Enabled=True, ..., FAICount=1
   [SHOT_0_CAM] (CameraSlaveParam/ShotConfig 필드 — ParamBase.Load 자동)
   [SHOT_0_FAI_0] FAIName=A1, MeasurementCount=2
   [SHOT_0_FAI_0_MEAS_0] Type=PointToLineDistance, DatumRef=B, ... (MeasurementBase 필드 — ParamBase.Load)
   [SHOT_0_FAI_0_MEAS_1] Type=EdgePairDistance, ...
   ```
   
   구현:
   - parentSequence를 InspectionSequence로 캐스팅 → seq
   - `[FIXTURE]` 섹션에서 `DisplayName` 로드 → `seq.DisplayName = ...`
   - `DatumCount` 읽기 → 루프에서 `[FIXTURE_DATUM_{d}]` 섹션:
     - `seq.AddDatum()` → datum
     - `datum.Load(loadFile, $"FIXTURE_DATUM_{d}")` (ParamBase.Load로 DatumConfig의 모든 프로퍼티 자동 로드)
   - `[SHOTS]` 섹션에서 `Count` 읽기 → 루프에서 `[SHOT_{s}]` 섹션:
     - `AddShot()` → shot (기존 메서드 활용, Datum 생성 코드 제거됨)
     - `shot.ShotName = loadFile["SHOT_{s}"]["ShotName"]`
     - `shot.Load(loadFile, $"SHOT_{s}_CAM")` (CameraSlaveParam 필드 자동 로드 — 조명 필드 포함)
     - `FAICount` 읽기 → 루프에서 `[SHOT_{s}_FAI_{f}]` 섹션:
       - `shot.AddFAI(faiName)` → fai
       - `fai.Load(loadFile, $"SHOT_{s}_FAI_{f}")` (FAIConfig 기본 필드 자동 로드)
       - `MeasurementCount` 읽기 → 루프에서 `[SHOT_{s}_FAI_{f}_MEAS_{m}]` 섹션:
         - `typeName = loadFile[measSection]["Type"]`
         - `var meas = MeasurementFactory.Create(typeName, fai)` → null이면 로그 + skip (T-06-01)
         - `meas.Load(loadFile, measSection)` (ParamBase.Load로 파생 클래스 필드 자동 로드)
         - `fai.Measurements.Add(meas)`

4. **SavePhase6Format(IniFile saveFile) 구현:**
   - Save 메서드에서 호출 (기존 Save를 SavePhase6Format으로 교체)
   - `[FORMAT]` 섹션: `Version=6`
   - `[FIXTURE]` 섹션: `DisplayName`, `DatumCount`
   - 각 DatumConfig: `datum.Save(saveFile, $"FIXTURE_DATUM_{d}")` (ParamBase.Save로 자동)
