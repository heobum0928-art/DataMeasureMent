## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| INI recipe file → RoiDefinition (JSON) | User-editable recipe files on disk deserialized via DataContractJsonSerializer. A user/operator with file-system access could author a RoiDefinition with arbitrary Radius/Center values. |
| MainResultViewerControl mouse input → Circle draft state | Trusted (local WPF UI thread). Cursor events originate from OS-level input; no external source. |
| HalconDisplayService draw calls → HWindow | Trusted (local process, Halcon native window). No external data injected here. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-01-01 | Tampering | RoiDefinition JSON deserialization (recipe file) | accept | Operator-only file-system access is assumed per project trust model (single-user industrial PC, no multi-tenant). Malformed Shape enum values fall back to default RoiShape.Rect by DataContract behavior — no crash path. Radius/Center are doubles; downstream Halcon DispCircle on negative/zero radius is guarded by `radius <= 0` early return in RenderCircleDraft. |
| T-11-01-02 | Denial of Service | MainResultViewerControl HMouseMove floods Render() during drag | mitigate | Existing Render() path already handles rapid MouseMove for Rect drag (L452-466) without throttling — same rate is acceptable for Circle. No new threat vector. |
| T-11-01-03 | Denial of Service | HOperatorSet.DispCircle with NaN/Infinity center/radius | mitigate | RenderCircleDraft guards `radius <= 0`; committed Circle branch in Render() inherits the same guard via `roi.Radius` (zero = nothing drawn, no exception). HDisplayService outer try/catch swallows remaining Halcon exceptions per existing pattern (L321-366). |
| T-11-01-04 | Information Disclosure | Circle Radius/Center logged or echoed | accept | Non-sensitive geometry data. No PII. Existing Rect ROI is treated the same. |
| T-11-01-05 | Elevation of Privilege | N/A (pure drawing primitives) | accept | No privilege boundary crossed. No process spawn, no shell, no network. |
