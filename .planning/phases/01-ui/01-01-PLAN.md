Create `WPF_Example/UI/ViewModel/InspectionViewModel.cs`. Namespace `ReringProject.UI`. Allman brace style. C# 7.2 only.

**Constructor:** Takes `InspectionRecipeManager recipeManager`, stores as `private readonly InspectionRecipeManager _recipeManager`. Initializes `_shots` ObservableCollection by wrapping each `_recipeManager.Shots` item in `ShotNodeViewModel`.

**Properties:**
- `ObservableCollection<ShotNodeViewModel> Shots { get; }` (readonly, set in constructor)
- `object SelectedNode { get; set; }` — setter calls `RaisePropertyChanged("SelectedNode")` then calls private `OnSelectionChanged()`. Type is `object` because WPF TreeView SelectedItem is object.
- `ObservableCollection<FAIResultRow> SelectedShotFAIResults { get; set; }` — setter calls `RaisePropertyChanged("SelectedShotFAIResults")`
- `ShotNodeViewModel SelectedShot { get; private set; }` — the currently selected Shot (resolved from SelectedNode)

**Private method `OnSelectionChanged()`:**
- If `SelectedNode is ShotNodeViewModel shotVm`: set `SelectedShot = shotVm`, rebuild `SelectedShotFAIResults` from `shotVm.ShotConfig.FAIList` (wrap each in `new FAIResultRow(fai)`)
- If `SelectedNode is FAINodeViewModel faiVm`: find parent ShotNodeViewModel from `Shots` collection, set `SelectedShot` to parent, rebuild results same way
- If `SelectedNode` is null: set `SelectedShot = null`, set `SelectedShotFAIResults` to empty collection

**CRUD methods (per D-08, D-09, D-10):**

`public void AddShot()`:
- Call `TextInputBox.Show("Shot 이름 입력", "SHOT_" + _recipeManager.Shots.Count, out string name)`
- If user cancelled, return
- Call `ShotConfig newShot = _recipeManager.AddShot(name)`
- Add `new ShotNodeViewModel(newShot)` to `Shots`

`public void AddFAIToSelectedShot()`:
- Determine target Shot: if `SelectedNode is ShotNodeViewModel shotVm` use shotVm; if `SelectedNode is FAINodeViewModel faiVm` find parent ShotNodeViewModel in Shots
- If no target shot found, return
- Call `TextInputBox.Show("FAI 이름 입력", "FAI_" + targetShotVm.ShotConfig.FAIList.Count, out string name)`
- If user cancelled, return
- Call `FAIConfig newFai = targetShotVm.ShotConfig.AddFAI(name)`
- Add `new FAINodeViewModel(newFai)` to `targetShotVm.FAIItems`
- If this shot is currently selected, rebuild `SelectedShotFAIResults`

`public void RemoveSelected()` (per D-10):
- If `SelectedNode is ShotNodeViewModel shotVm`:
  - Call `CustomMessageBox.ShowConfirmation("Shot 삭제", $"Shot 삭제: \"{shotVm.Name}\" Shot과 하위 FAI를 모두 삭제합니다. 계속하시겠습니까?", MessageBoxButton.YesNo)`
  - If result != Yes, return
  - Find index in `Shots`, call `_recipeManager.RemoveShot(index)` and `Shots.RemoveAt(index)`
  - Set `SelectedNode = null`
- If `SelectedNode is FAINodeViewModel faiVm`:
  - Find parent ShotNodeViewModel in Shots
  - Call `CustomMessageBox.ShowConfirmation("FAI 삭제", $"FAI 삭제: \"{faiVm.Name}\" FAI를 삭제합니다. 계속하시겠습니까?", MessageBoxButton.YesNo)`
  - If result != Yes, return
  - Find index in parent.FAIItems, call `parentShot.ShotConfig.RemoveFAI(index)` and `parentShot.FAIItems.RemoveAt(index)`
  - Set `SelectedNode = null`

`public void RenameSelected()` (per D-09 — Name only):
- If `SelectedNode is ShotNodeViewModel shotVm`:
  - Call `TextInputBox.Show("Shot 이름 수정", shotVm.Name, out string newName)`
  - If user cancelled, return; set `shotVm.Name = newName`
- If `SelectedNode is FAINodeViewModel faiVm`:
  - Call `TextInputBox.Show("FAI 이름 수정", faiVm.Name, out string newName)`
  - If user cancelled, return; set `faiVm.Name = newName`

**Helper properties for toolbar button states:**
- `public bool CanRemove => SelectedNode != null` (for Remove button IsEnabled binding)
- `public bool CanRename => SelectedNode != null` (for Edit button IsEnabled binding)
- In `OnSelectionChanged()`, also call `RaisePropertyChanged("CanRemove")` and `RaisePropertyChanged("CanRename")`

Add XML doc comment on the class: `/// <summary>Root ViewModel for Shot/FAI inspection view. Coordinates TreeView, canvas, and result table.</summary>`
