**Main method** per D-15:
```csharp
//260409 hbk Phase 4: Datum finding service (D-15, D-16)
public bool TryFindDatum(HImage image, DatumConfig config, out HTuple transform, out string error)
```

Implementation steps per D-16:

1. **Validate inputs:**
   - If `config == null || !config.IsConfigured` → set `transform` to identity (`HOperatorSet.HomMat2dIdentity`), `error = null`, return `true` (D-08 fallback: unconfigured = identity pass-through)

2. **Find Line 1 (reference X-axis):**
   - Create measure rectangle: `HOperatorSet.GenMeasureRectangle2(config.Line1_Row, config.Line1_Col, config.Line1_Phi, config.Line1_Length1, config.Line1_Length2, imageWidth, imageHeight, "nearest_neighbor", out measureHandle1)`
   - Execute: `HOperatorSet.MeasurePos(image, measureHandle1, config.Sigma, config.EdgeThreshold, config.EdgePolarity, "all", out rowEdge1, out colEdge1, out amp1, out dist1)`
   - Close measure handle immediately after use
   - If fewer than 2 edge points → error = "Line1: insufficient edge points", return false
   - Fit line: `HOperatorSet.GenContourPolygonXld(out contour1, rowEdge1, colEdge1)` then `HOperatorSet.FitLineContourXld(contour1, "tukey", -1, 0, 5, 2, out line1RowBegin, out line1ColBegin, out line1RowEnd, out line1ColEnd, out nr1, out nc1, out dist1)`
   - Dispose contour

3. **Find Line 2 (reference Y-axis):** Same as above with Line2 ROI fields.

4. **Compute intersection** per D-03:
   - `HOperatorSet.IntersectionLl(line1RowBegin, line1ColBegin, line1RowEnd, line1ColEnd, line2RowBegin, line2ColBegin, line2RowEnd, line2ColEnd, out curRow, out curCol, out isOverlapping)`
   - If `isOverlapping == 1` (parallel lines) → error = "Lines are parallel, no intersection", return false

5. **Compute current angle:**
   - `double curAngle = Math.Atan2(line1RowEnd.D - line1RowBegin.D, line1ColEnd.D - line1ColBegin.D);`

6. **Compute transform** per D-07 step 3:
   - `double dRow = curRow.D - config.RefOriginRow;`
   - `double dCol = curCol.D - config.RefOriginCol;`
   - `double dAngle = curAngle - config.RefAngleRad;`
   - Build hom_mat2d:
     ```csharp
     HOperatorSet.HomMat2dIdentity(out transform);
     HOperatorSet.HomMat2dTranslate(transform, dRow, dCol, out transform);
     HOperatorSet.HomMat2dRotate(transform, dAngle, curRow.D, curCol.D, out transform);
     ```

7. Return `true`, `error = null`.

Wrap all Halcon calls in `try { } catch (Exception ex) { error = ex.Message; transform = identity; return false; }`.

**Helper method for teaching** per D-13:
```csharp
//260409 hbk Phase 4: store reference datum from teaching image (D-13)
public bool TryTeachDatum(HImage image, DatumConfig config, out string error)
```
- Runs steps 1-4 above (find lines, compute intersection and angle)
- On success: stores `config.RefOriginRow = curRow.D`, `config.RefOriginCol = curCol.D`, `config.RefAngleRad = curAngle`, `config.IsConfigured = true`
- Returns true/false
