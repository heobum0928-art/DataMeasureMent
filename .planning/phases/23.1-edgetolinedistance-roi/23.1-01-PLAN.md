`FitLineContourXld` 는 라인 피팅에 최소 2개 에지점을 요구한다. `EdgeSelection="First"` 또는 `"Last"` 는 `MeasurePos` 가 단일 에지점 1개만 반환하게 하므로 라인 피팅이 항상 실패한다. commit 898a96f 는 default 값만 `First→All` 로 교정했으나, 레거시 INI 에 `EdgeSelection=First` 가 저장돼 있거나 사용자가 PropertyGrid 에서 다시 First/Last 를 고르면 결함이 재발한다.
