### Test 1: msbuild Debug/x64 Rebuild (자동)

- **목적**: Plan 01 변경 후 빌드 무결성 확인.
- **명령 (PowerShell — 환경 비의존, vswhere fallback 포함)**:
  ```powershell
  cd "C:\Info\Project\DataMeasurement"
  $msbuild = (Get-Command msbuild.exe -ErrorAction SilentlyContinue).Source
  if (-not $msbuild) {
      $vswhere = "${env:ProgramFiles(x86)}\Microsoft Visual Studio\Installer\vswhere.exe"
      if (Test-Path $vswhere) {
          $msbuild = & $vswhere -latest -requires Microsoft.Component.MSBuild `
                      -find "MSBuild\**\Bin\MSBuild.exe" | Select-Object -First 1
      }
  }
  if (-not $msbuild) {
      throw "MSBuild not found. msbuild.exe 를 PATH 에 추가하거나 VS Build Tools 설치 필요."
  }
  & $msbuild WPF_Example\DatumMeasurement.csproj /t:Rebuild /p:Configuration=Debug /p:Platform=x64 /nologo /v:minimal `
      2>&1 | Tee-Object -FilePath $env:TEMP\build.log
  Write-Host "Exit: $LASTEXITCODE"
  (Select-String -Path $env:TEMP\build.log -Pattern "error " | Measure-Object).Count
  (Select-String -Path $env:TEMP\build.log -Pattern "warning " | Measure-Object).Count
  ```
- **명령 (Bash 대체 — msbuild 가 PATH 에 있을 때)**:
  ```bash
  cd "C:/Info/Project/DataMeasurement"
  MSBUILD="${MSBUILD_PATH:-$(where msbuild.exe 2>/dev/null | head -1)}"

  "$MSBUILD" WPF_Example/DatumMeasurement.csproj //t:Rebuild //p:Configuration=Debug //p:Platform=x64 //nologo //v:minimal 2>&1 | tee /tmp/build.log
  echo "Exit: $?"; grep -c "error " /tmp/build.log; grep -c "warning " /tmp/build.log
  ```
