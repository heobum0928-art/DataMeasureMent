## 부호 공식 유도 메모 + W1 정규화 동치 증명

### useAngle2 분기 부호 공식 유도

axis 방향벡터: `d = (sinθ2, cosθ2)` (measureX useAngle2 분기의 axisR1..C2 정의에서 시작점→끝점 방향)

axis 의 우측 법선 (시계 90° 회전): `n_right = (cosθ2, −sinθ2)`

부호 의도: `+X` (이미지 col 증가 방향) 양수.

footRow/footCol 은 projection_pl 결과로 axis 위의 수선의 발. (Δrow, Δcol) = (pRow − footRow, pCol − footCol) 는 axis 에 수직 (footPerp).

부호 = `Δrow * n_right_row + Δcol * n_right_col = Δrow * cosθ2 − Δcol * sinθ2`

→ `signedPx = (pRow − footRow) * cosT − (pCol − footCol) * sinT` (sinT/cosT 는 cosT≥0 정규화 후 angleSrc=DatumAngle2Rad 값)

### W1: 정확한 직각 케이스 정규화 후 동치 증명 (UAT 시나리오 2)

가정: 부품이 정확한 직각 → `θ2 = θ + π/2` (θ = DatumAngleRad, θ2 = DatumAngle2Rad).

cosT≥0 정규화 가드 후 angleSrc:
- useAngle2: `sinθ2 = sin(θ+π/2) = cosθ`, `cosθ2 = cos(θ+π/2) = −sinθ`
- cosθ2 < 0 시 정규화: `sinθ2 ← −cosθ`, `cosθ2 ← sinθ` (cosT≥0)
- 즉, useAngle2 정규화 후 (sinT, cosT) = (−cosθ_orig, sinθ_orig) (θ_orig 도 동일 cosT≥0 정규화 후의 값)

폴백 분기 정규화 후 (sinT, cosT) = (sinθ_orig, cosθ_orig). 단, axis 정의가:
- useAngle2: axisR1 = origin − 200*sinT_a, axisC1 = origin − 200*cosT_a (sinT_a, cosT_a = useAngle2 정규화값)
- 폴백: axisR1 = origin − 200*cosT_f, axisC1 = origin + 200*sinT_f (sinT_f, cosT_f = 폴백 정규화값)

θ2 = θ + π/2 → useAngle2 정규화 (sinT_a, cosT_a) = (−cosθ_orig, sinθ_orig). 폴백 (sinT_f, cosT_f) = (sinθ_orig, cosθ_orig).

axisR1 useAngle2: `origin − 200*(−cosθ_orig) = origin + 200*cosθ_orig`.
axisR1 폴백: `origin − 200*cosθ_orig`.

방향이 반대지만 axis 는 라인 (방향성 무관 — projection_pl 는 같은 직선이면 동일 수선의 발 반환). 따라서 footRow/footCol 은 두 분기에서 동일.

signedPx useAngle2: `(Δrow)*cosT_a − (Δcol)*sinT_a = Δrow*sinθ_orig − Δcol*(−cosθ_orig) = Δrow*sinθ + Δcol*cosθ`
signedPx 폴백: `(Δrow)*sinT_f + (Δcol)*cosT_f = Δrow*sinθ + Δcol*cosθ`

→ **수학적으로 동치 (Δ1 = 0)**. 따라서 정확한 직각 케이스 (시나리오 2) 회귀 0 보장.
