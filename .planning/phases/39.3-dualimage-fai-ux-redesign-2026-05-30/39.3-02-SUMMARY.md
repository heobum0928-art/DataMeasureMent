# Phase 39.3 Plan 02: DualImage Swap UX Wiring (D-G2) Summary

## Objective

Phase 34.1 / 36 / 37 에서 Side Datum DualImage 를 위해 구축한 swap UX 인프라 (`btn_swapHorizontal`/`btn_swapVertical` 토글, `border_imageSourceBadge`, `_currentImageSource` enum, `UpdateImageSourceBadge` 헬퍼, `BadgeBrushHorizontal`/`BadgeBrushVertical` frozen brush) 를 그대로 재사용하면서, Measurement DualImage 노드에도 동일한 swap 동작이 가능하도록 wiring 5 분기 (E, F, I, G, H) 를 추가. **Datum 와 Measurement 가 swap 인프라를 공유하지만 동시 활성은 불가능** — `_selectedDatumForSwap` 와 `_selectedDualImageMeasurement` 가 양방향 mutex 로 항상 한쪽만 non-null.
