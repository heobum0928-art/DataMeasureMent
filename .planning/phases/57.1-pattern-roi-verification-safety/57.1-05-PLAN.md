# 57.1-05 — 패턴 ROI 그리기 버튼 진행 확인(Yes/No) 추가

<objective>
Datum 티칭 존(유효한 Datum 노드 선택 상태)에서도 "패턴 1 / 패턴 2 ROI 그리기" 버튼을 무심코 클릭하는 것을 막기 위해, 실제 그리기 모드 진입 전에 진행 여부 확인(OK/Cancel) 메시지박스를 띄운다. 사용자가 [취소]하면 아무 동작 없이 종료.

현 상태: Datum 노드가 선택돼 있으면 버튼 클릭 즉시 그리기 모드로 진입(확인 없음) → 무심코 누르면 기존 패턴 ROI 가 덮일 수 있음.
</objective>
