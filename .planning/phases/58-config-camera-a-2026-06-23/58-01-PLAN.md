        }

        private const int PC_ROLE_DEFAULT = 1; //260622 hbk Phase 48

        partial void AfterLoad()
        {
            RestorePcRoleDefault();
        }

        private void RestorePcRoleDefault()
        {
            bool bPcRoleMissing = PcRole == 0;
            if (bPcRoleMissing)
            {
                PcRole = PC_ROLE_DEFAULT;
            }
        }
    }
}

INI serialization rule (framework WPF_Example/Setting/SystemSetting.cs Load(), reflection-based):
- A public property with `[Category("ETHERNET_VISION")]` is auto-mapped to INI group "ETHERNET_VISION", key = property name.
- Supported types in the Load/Save switch: Int32, Double, String, Boolean (plus Rect/Line/Circle/ModelFinder). **enum is NOT a supported case** — that is why an int-backing property is required (CameraRoleValue precedent).
- Missing keys load as 0 / "" / false (reference_parambase_missing_key_zeroes_default.md) — hence AfterLoad() restoration for the 8.652 default.
