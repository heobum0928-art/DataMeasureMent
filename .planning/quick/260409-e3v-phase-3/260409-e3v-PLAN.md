  <name>Task 2: Rewrite FAIEdgeMeasurementService with sample-strip line-fitting algorithm</name>
  <files>WPF_Example/Halcon/Algorithms/FAIEdgeMeasurementService.cs</files>
  <action>
Complete rewrite of FAIEdgeMeasurementService.cs based on MeasurementAlgorithm.TryInspectSingleEdgeInternal pattern. Keep `bool TryMeasure(HImage image, FAIConfig fai, out FAIEdgeMeasurementResult result)` signature unchanged.

**Delete entirely:** SelectEdgeIndices method, CalculateMmDistance method, current TryMeasure body, BuildOverlays method.

**New TryMeasure implementation:**

1. **Validation:** null checks on image/fai, ROI_Length1/Length2 > 0 check (same as current)

2. **Setup from FAIConfig:**
   ```
   scanHorizontal = EdgeDirection is "LtoR" or "RtoL" (case-insensitive)
   measurePhi = LtoR->0, RtoL->PI, TtoB->-PI/2, BtoT->PI/2
   sampleCount = Math.Max(1, fai.EdgeSampleCount)
   sigma = Math.Max(0.4, fai.Sigma)
   threshold = Math.Max(1, fai.EdgeThreshold)
   polarity = EdgePolarity=="LightToDark" ? "negative" : "positive"
   ```

3. **Compute ROI bounding box** from FAIConfig Rectangle2 params (same sin/cos logic as ToRoiDefinition to get top/left/bottom/right)

4. **Branch on EdgeSelection:**

   **Case "Both":**
   - Create two point collections: firstEdgeRows/Cols, lastEdgeRows/Cols
   - Loop sampleCount strips (same strip logic as MeasurementAlgorithm):
     - GenRectangle1 for strip -> SmallestRectangle2 -> GenMeasureRectangle2 with measurePhi -> MeasurePos with select="all", polarity from config
     - CloseMeasure handle in finally block for each strip
     - If edgeCount >= 2: add rows[0]/cols[0] to firstEdge*, add rows[last]/cols[last] to lastEdge*
     - If edgeCount == 1: add to firstEdge* only
   - Apply TrimExtremePoints to firstEdge* (using fai.EdgeTrimCount)
   - Apply TrimExtremePoints to lastEdge* (using fai.EdgeTrimCount)
   - If either collection has <= 1 point: return false
   - FitLineContourXld on firstEdge* -> line1 (row1,col1,row2,col2)
   - FitLineContourXld on lastEdge* -> line2 (row1,col1,row2,col2)
   - Calculate perpendicular distance between two parallel lines:
     ```
     // Use line1 direction vector, project line2 midpoint onto line1 normal
     double dx = line1Col2 - line1Col1;
     double dy = line1Row2 - line1Row1;
     double len = Math.Sqrt(dx*dx + dy*dy);
     if (len < 1e-9) return false;
     // Normal vector (perpendicular to line1)
     double nx = -dy / len;
     double ny = dx / len;
     // Midpoint of line2
     double midRow = (line2Row1 + line2Row2) / 2.0;
     double midCol = (line2Col1 + line2Col2) / 2.0;
     // Midpoint of line1
     double refRow = (line1Row1 + line1Row2) / 2.0;
     double refCol = (line1Col1 + line1Col2) / 2.0;
     // Signed distance = dot product of (mid2-mid1) with normal
     double pixelDist = Math.Abs(nx * (midCol - refCol) + ny * (midRow - refRow));
     ```
   - Convert to mm: use PixelResolutionX if scanHorizontal, PixelResolutionY if vertical (same axis logic as scan direction is perpendicular to edge direction)
     Actually: distance is perpendicular to the fitted line. For horizontal scan (LtoR/RtoL), edges are vertical lines, distance between them is horizontal -> use PixelResolutionX.
     For vertical scan (TtoB/BtoT), edges are horizontal lines, distance between them is vertical -> use PixelResolutionY.
     ```
     double mmDist = scanHorizontal ? pixelDist * fai.PixelResolutionX : pixelDist * fai.PixelResolutionY;
     ```

   **Case "First" or "Last":**
   - Single point collection: allRows/allCols
   - Loop sampleCount strips (same as MeasurementAlgorithm):
     - GenMeasureRectangle2 -> MeasurePos with select = EdgeSelection=="Last" ? "last" : "first"
     - CloseMeasure handle in finally
     - Collect detected edge points
   - Apply TrimExtremePoints
   - FitLineContourXld -> single fitted line
   - Set result: DistancePixel=0, DistanceMm=0 (single edge, no distance)

5. **Build result:**
   - Set Line1* fields from first fitted line
   - Set Line2* fields from second fitted line (zeros if single edge)
   - Edge1Row/Col = midpoint of line1, Edge2Row/Col = midpoint of line2
   - EdgePointCount = total points before trim
   - Build overlays: one overlay per fitted line (using line endpoints), plus distance line if Both

6. **Copy TrimExtremePoints** as private static method from MeasurementAlgorithm (identical logic).

7. **Error handling:** Wrap entire body in try/catch returning false. Each strip's GenMeasureRectangle2 handle must be closed in a per-strip finally. Use the pattern:
   ```
   HTuple handle = null;
   try {
       HOperatorSet.GenMeasureRectangle2(..., out handle);
       HOperatorSet.MeasurePos(...);
   } finally {
       if (handle != null) try { HOperatorSet.CloseMeasure(handle); } catch { }
   }
   ```

All lines marked `//260409 hbk`. Allman brace style. C# 7.2 only (no `is` pattern matching, no `??=`).
