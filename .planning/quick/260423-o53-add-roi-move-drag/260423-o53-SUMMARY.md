# Quick Task 260423-o53: ROI 이동 드래그 기능 — SUMMARY

## 완료 내용

선택된 Rect/Circle ROI를 마우스 드래그로 이동하는 기능 추가.

## 파일 변경

- `WPF_Example/UI/ContentItem/MainResultViewerControl.xaml.cs`
  - `RoiMoveCompletedArgs` 클래스 추가 (RoiId, DeltaRow, DeltaCol)
  - `RoiMoveCompleted` 이벤트 추가
  - 이동 상태 필드: `_isMovingRoi`, `_moveStartImagePoint`, `_movingRoiSnapshot`
  - `HitTestSelectedRoi(Point)` — 선택 ROI 내부 클릭 판정 (Circle: 거리 ≤ 반지름, Rect: bbox contains)
  - `IsAnyDrawingModeActive()` — 드로잉 모드(Rect/Circle/Polygon/Calibration) 진입 중 이동 차단
  - `HMouseDown`: 드로잉 모드 아니고 선택 ROI 내부 클릭 → 이동 시작, snapshot 저장, 커서 SizeAll
  - `HMouseMove`: 이동 중이면 snapshot + delta로 `_rois` 해당 항목 좌표 갱신 + Render
  - `HMouseUp`: 이동 종료 시 델타 계산 → `RoiMoveCompleted` 발생 (움직임 ≥ 0.5px일 때만)

- `WPF_Example/UI/ContentItem/MainView.xaml.cs`
  - 생성자에서 `RoiMoveCompleted` 구독
  - `MainView_Unloaded`에서 구독 해제
  - `HalconViewer_RoiMoveCompleted` 핸들러: `FindFAIByName(RoiId)` → FAI에 `CircleDiameterMeasurement`이 있으면 `Circle_Row/Col`에 델타 적용, 없으면 `ROI_Row/Col`에 적용 → `UpdateDisplayState` 재렌더

## 검증

- `msbuild /p:Configuration=Debug /p:Platform=x64` 빌드 성공 (기존 경고 외 신규 경고 없음)
- 런타임 UAT 대기

## UAT 체크리스트

- [ ] Rect ROI 생성 → FAI 선택 → 내부 드래그 시 이동
- [ ] Circle ROI 생성 → 선택 → 내부 드래그 시 이동
- [ ] 드로잉 모드(Rect/Circle 버튼 ON) 중에는 이동 차단
- [ ] 이동 후 `UpdateDisplayState` 재갱신해도 새 위치 유지 (모델 반영 확인)
- [ ] 기존 드로잉 기능 회귀 없음
