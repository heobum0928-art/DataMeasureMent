HALCON projection_pl (HALCON 24.11 공식 문서 확인 완료, HalconDotNet 이미 import):
  HOperatorSet.ProjectionPl(HTuple row, HTuple column,
                            HTuple row1, HTuple column1, HTuple row2, HTuple column2,
                            out HTuple rowProj, out HTuple colProj)
  점 (row,column) 을 직선 (row1,column1)-(row2,column2) 에 정사영 → 수선의 발
  (rowProj,colProj) 반환. 인자 순서 = 점 먼저, 직선 2점, 출력 2개. AffineTransPoint2d
  와 동일하게 out HTuple → .D 로 double (현 파일 L131-133 패턴 동일).

각도 규약: 이 코드베이스의 각도 = Atan2(Δrow, Δcol) → 각도 α 의 방향벡터 = (sinα, cosα).
  datum 수평선(x축) = 각도 θ → 방향 (sinθ, cosθ).
  datum 수직선(y축) = 각도 θ+90° → 방향 (sin(θ+90°), cos(θ+90°)) = (cosθ, -sinθ).
