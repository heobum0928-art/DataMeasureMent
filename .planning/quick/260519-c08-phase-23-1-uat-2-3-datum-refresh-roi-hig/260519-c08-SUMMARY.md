- **afb03d3:** `FAIName_` 접두사 매칭 도입 → FAI 선택 시 자식 ROI 전부 노란색. 동작 확인.
- **33212cd:** 결과 리스트박스 행 선택이 `selectedRow.FAIName` 만 써 FAI 노드 선택과 동일 selId → 색 안 변함. 복합 Id `FAIName_측정명` 으로 그 측정 ROI 만 하이라이트.
