# Phase 260723-c98: Re-expose MeasCorrectionFactor via MeasCorrectionEnabled Opt-In Summary

## Accomplishments

- `MeasurementBase.MeasCorrectionFactor` is visible in the PropertyGrid again (previously hidden via `[Browsable(false)]` since 2026-07-15's introducing commit `dcfccc7`).
- New `MeasCorrectionEnabled` (bool, default `false`) property added ŌĆö correction only applies when explicitly turned on, in addition to the existing `AppliesCorrectionFactor` per-type gate.
- `EvaluateJudgement`'s gate is now `MeasCorrectionEnabled && AppliesCorrectionFactor && MeasCorrectionFactor > 0.0` ŌĆö a pure additive AND over the previous two-condition gate.
